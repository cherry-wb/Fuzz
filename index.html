Fuzz

jQuery plugin to create background image noise using the canvas element.

Fuzz? What does it do?

Fuzz randomly generates a background image noise effect on a web page using the canvas element. No need for Photoshop and images (assuming you have a canvas capable browser...!).

You can set your background using CSS and apply the noise after.

Requirements

  • Canvas-capable browser
  • jQuery JavaScript library included on the page

Usage

Add the plugin and jQuery to your HTML file. Call the plugin like so:

e.g: $('body').fuzz();

If you have sections of your page with different backgrounds you'll need to put these in your jQuery selector. For example, for this page, the header and footer have a different background colour, so I need to use:

$('body, header, footer').fuzz();

The options

  • color: true (for colour noise) or false (for greyscale noise)
  • size: 0 to 500. Refers to the size of the generated image. This defaults to 100, which generates a 100x100 pixel image. This should be large enough to not notice repeats, but small enough that it doesn't take too long to draw. The limit of 500 is fairly arbitrary, when I tested this, it took around 1.2 seconds to run the function with a 500x500 pixel image, which I think is probably actually too long to add to your page load time really. With a 100x100 pixel image, it takes around 50ms to run the function, so should be satisfactorily unnoticeable.
  • opacity: 0 to 1. Refers to the opacity of the generated image, or essentially, how much noise you see on the page. This defaults to 0.075, which applies a very subtle amount of noise.
  • The HTML isn't pretty (if that matters)

Notes/future steps

  • In theory, it should be possible to layer the noise on top of an existing background image with CSS3 multiple backgrounds. The plugin does check for the existence of background-image properties, and if any do exist, adds the noise without overwriting them. The resulting markup looks right but for some reason it doesn't actually work. So if you want noise over existing background images, treat it like CSS2 where you have one background per element.
  • At the moment, the greyscale noise is just grey but I would like to add an option where you can have noise in a particular colour.
  • I tried in vain to use the British spelling for 'colour' in the plugin but unfortunately it just confused me. So it's 'color'.
