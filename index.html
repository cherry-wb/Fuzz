Background noise generated with canvas

Functionality

  • Generates a background image noise effect using canvas
  • Specify the amount of colour/saturation (0 = totally bw, 1 = totally colour)
  • Specify opacity (0 = invisible, 1 = all noise. Around 0.1-0.2 is recommended.)
  • Specify different options for different elements so you can have different noise for header/footer/main etc

Requirements

  • Canvas capable browser
  • jQuery

Usage

Add the plugin to your HTML file. Optional declare some arguments.
